## 📋 about

A photo-sharing SPA where users add cards with place photos, like and delete them, and edit their profile — built with vanilla JavaScript and connected to a REST API.

<!-- ![demo](link-to-screenshot) -->

## ✨ features

- 🖼️ Add and delete photo cards
- ❤️ Like / unlike cards
- 👤 Edit profile name, bio and avatar
- ✅ Client-side form validation
- 🌐 REST API integration
- 📱 Responsive layout

## 🛠️ tech stack

- Vanilla JavaScript ES6+
- HTML5 + CSS3 (BEM)
- REST API

## 🚀 getting started

```bash
git clone https://github.com/zeawale/mesto-project-ff.git
cd mesto-project-ff
# open index.html in browser or use live server
```
